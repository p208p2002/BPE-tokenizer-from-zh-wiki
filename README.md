### 與 GPT2-Chinese (Wenzhong) 比較
Wenzhong模型雖然使用中文語料訓練，可是並沒有針對中文語料建立模型詞表，
雖然依靠BPE演算法可使其 back-off bytes (避免oov)，但因編碼長度變長導致效率較差，並且在令牌化後缺失語義。
